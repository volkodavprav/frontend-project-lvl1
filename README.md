# Welcome to Brain Games

Brain Games is set of fun games. To win the game you need to answer correctly 3 times in a row.

### Installation
Make sure you have Node 16+ installed.

1. Clone repository
2. Run `make install`
3. Run the game using its name e.g: `brain-even`
